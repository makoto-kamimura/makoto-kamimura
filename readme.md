## 私について / About Me

- 🧠 **シングルタスクエンジニア** — 一度にひとつのことを、深く片付けるスタイル
- 🖥️ **端末に依存しない開発環境**を好みます（Docker / DevContainer / Web IDE 中心）  
  *Prefers a development environment that doesn't depend on any specific terminal or machine.*
- 🛠️ Web アプリ開発と DevOps が軸。最近は LLM 活用と脆弱性診断の自動化に関心
- 📦 開発スタイル: Docker で再現性を担保し、最小依存で堅牢に動かす
- 🌲 拠点: リモート中心（時々キャンプ場から）

### 現在の関心領域

- **LLMOps** — Dify / LangChain を使った業務アプリの試作
- **セキュリティ自動化** — 脆弱性診断ツールの Web UI 化
- **ポータブルな開発環境** — どの端末からでも同じ手順で立ち上がる構成
